# Docker Ubuntu HTML Hello World

Bu repo, Docker kullanarak Ubuntu tabanlı bir container içinde
basit bir HTML "Hello World" uygulaması çalıştırmak için hazırlanmıştır.

## Kullanılan Teknolojiler
- Docker
- Ubuntu 22.04
- HTML
